### 2.3 논문에서 기술한 JSON 예시와의 구조 비교

논문 3장 §3.2에 제시된 RASE JSON 예시:
```json
{
  "S": {
    "conditions": [
      { "height_range": "높이 10m 이하", "rule": "1.5m 이상 이격" },
      { "height_range": "높이 10m 초과", "rule": "높이 × 0.5 이상 이격" }
    ]
  }
}
```

실제 생성된 JSON은 이 구조를 확장하여 `parameters` 블록(수치 분리)과 `rule_type` 필드(fixed/proportional 구분)를 추가하였다. 논문 예시는 개념 설명용 약식이므로, 실제 구현이 이를 확장한 것은 **학술적으로 문제없다**. 다만, 논문에 "실제 사용된 JSON은 parameters 블록을 추가하여 수치를 분리하였다"는 보충 설명을 추가하면 더 정확하다.


## 3. 시나리오 보고서 ↔ 논문 초고 정합성

### 3.1 필지 수 검증

| 출처 | 1종 | 2종 | 3종 | 합계 |
|------|------|-------|------|-------|
| 논문 표 3-4 (최종 분석 필지) | 2,477 | 11,420 | 7,383 | 21,280 |
| 보고서: 시나리오 B·C·D | 2,477 | 11,420 | 7,383 | 21,280 |
| 보고서: 시나리오 A | 2,303 | 10,635 | 6,793 | 19,731 |
| 보고서: 시나리오 E·F | 1,678 | 7,157 | 4,652 | 13,487 |

#### !! 불일치 발견 — 시나리오 A의 필지 수

**문제**: 논문 표 3-4는 "형상 오류 304필지 제외 후 21,280필지"라고 기술하지만, 시나리오 A의 실제 분석 필지는 19,731개로 1,549필지가 추가 누락되어 있다.

**원인 추정**: 시나리오 A는 중간 구간(mid_threshold) 없이 4층(12m)부터 바로 h×0.5 사선이 적용되어, 이격거리가 상대적으로 크다(12m에서 6.0m vs B의 5.0m). 소규모 필지에서 기하 연산(Boolean Difference)이 실패할 가능성이 B보다 높으며, 실패한 필지의 결과 행이 누락되어 필지 수가 줄어든 것으로 보인다.

**학술적 영향**: 
- B·C·D 간 비교(동일 21,280필지): **영향 없음** — 직접 비교 유효
- E·F 간 비교(동일 13,487필지): **영향 없음** — 직접 비교 유효  
- A ↔ B 절대값 비교: **주의 필요** — 필지 집합이 다르므로 직접 비교 시 유보 조건 필요

**권고**: 
1. 논문 4장 §2의 A-B 비교 표(표 4-2)에 "시나리오 A는 기하 연산 실패로 인해 분석 필지가 19,731개"임을 각주로 명시
2. 가능하다면 기하 연산 실패 시 0㎡로 채우는 보정(fill-zero) 적용 후 재실행하여 21,280필지로 통일

#### !! 불일치 발견 — 시나리오 E·F의 필지 수

**문제**: E·F의 필지 수가 13,487개로 B·C·D(21,280)의 63% 수준이다.

**원인 추정**: 시나리오 E는 저층부 이격이 2.0m로 확대되어 매우 소규모 필지에서 1층부터 기하 연산이 실패한다. 시나리오 F는 E와 동일한 배치(batch) 실행에서 생성되었을 가능성이 있어 동일한 필지 집합을 공유한다.

**학술적 영향**:
- E와 F의 상호 비교: **유효** (동일 필지 집합)
- E·F와 B·C·D 간 절대값 비교: **제한적** — 필지 구성이 다름

**권고**: 논문에 "E·F 시나리오의 분석 필지 수가 B·C·D와 다른 이유"를 명시하고, 시나리오 간 비교는 동일 필지 집합 내에서 수행함을 원칙으로 기술

### 3.2 논문 표 4-2 (A-B 비교) 검증

논문 표 4-2의 "4층 증가율 +2.0%(1종), +3.1%(2종), +1.8%(3종)"은 A와 B의 4층 평균면적 비교에서 산출된 값이다.

**보고서 데이터로 역산**:
- 1종 4층: A mean=1,072.64, B mean=1,039.26 → (1039.26-1072.64)/1072.64 = **-3.1%** (감소)

이 결과는 논문의 "+2.0% 증가"와 **모순**된다. 

**원인**: B의 평균면적(1,039.26)이 A(1,072.64)보다 작은 것은 필지 집합이 다르기 때문이다. B는 21,280필지(A에서 실패한 1,549필지 포함)를 분석하므로, 추가된 소규모 필지들이 평균을 하향시킨 것이다. 논문의 +2.0%는 동일 필지에서의 비교를 전제한 값으로, 현재 CSV의 집계 방식(전체 필지 평균)과는 다른 산출 기준을 사용했을 가능성이 있다.

**권고**: 
1. A-B 비교 시 "동일 필지에서의 면적 차이 합산" 방식과 "시나리오별 전체 평균 비교" 방식을 명확히 구분
2. 표 4-2의 증가율 산출 근거를 명시하거나, fill-zero 보정 후 21,280필지로 통일한 데이터로 재산출

### 3.3 시나리오 결과의 기하학적 타당성 검증

각 시나리오의 결과가 이격거리 규칙에 부합하는지 논리적으로 검증:

| 검증 항목 | 기대 | 실제 | 판정 |
|----------|------|------|------|
| B의 4~5층 면적 동일 | 둘 다 5.0m 고정 → 동일해야 함 | 142.50 = 142.50 | **일치** |
| C의 5층 < B의 5층 | C는 15m>14m → h×0.5=7.5m, B는 5.0m | 113.96 < 142.50 | **일치** |
| D의 6층 = D의 4~5층 | 18m ≤ 20m → 고정 5.0m | 142.50 = 142.50 | **일치** |
| B·C·D의 7층 동일 | 21m > 20m → 모두 h×0.5=10.5m | 78.51 = 78.51 = 78.51 | **일치** |
| E와 F의 4~5층 동일 | 둘 다 5.0m 고정 | 143.60 = 143.60 | **일치** |
| F의 6층 > E의 6층 | F: 7.2m < E: 9.0m → F가 더 넓음 | 120.79 > 100.92 | **일치** |
| E의 1~3층 < F의 1~3층 | E: 2.0m > F: 1.5m → E가 더 좁음 | 173.01 < 177.05 | **일치** |

**판정**: 모든 시나리오의 결과가 이격거리 규칙에 기하학적으로 부합한다. ✓

## 5. 학술적 활용 가능성 종합 판정

### 5.3 최종 판정

**시나리오 보고서의 B·C·D 비교(4장 §3.1) 및 E·F 비교(4장 §3.2, §3.3)는 동일 필지 집합 내에서의 비교이므로 학술적으로 문제없이 활용 가능하다.**

**A-B 비교(4장 §2)는 필지 수 차이를 논문에 명시하거나, fill-zero 보정 후 재실행하여 데이터 일관성을 확보하는 것이 권장된다.**

RASE JSON 6개 파일은 논문의 RASE 방법론 기술, 시나리오 정의, 파이프라인 설계 원리와 모두 정합하며, 4장의 "JSON 수치만 변경하여 시나리오 전환" 주장을 뒷받침하는 실증 자료로 활용 가능하다.
